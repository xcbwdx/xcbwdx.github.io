#战地1

##战地1

| 名称               | 载具                                                         | 介绍                                                         |
| ------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 吉普小车           | 在《战地1》里面有两种类型，一种是我们在战地里面经常看见的可乘坐三人后面携带一把机枪的小吉普，另一种是全副装甲的装甲小车。 |                                                              |
| 轻型坦克           | 支援型轻坦克                                                 | 具有强大的进攻能力，并且适合近距离到中距离的战斗，同时它与其他轻坦克不一样，可以从车后投掷一个大型的子弹医疗包支援友军。 |
| 轻型侧翼攻击坦克   | 非常适合于近距离作战的轻型坦克，不单止具有强大的对步兵火力，还能够在经过的路上部署反坦克地雷，加上它轻便快捷的特性，可以当作是一种新的反装甲玩法。 |                                                              |
| 轻型榴弹炮坦克     | 配备有榴弹炮的轻坦克，适合于对远距离或定点的敌人、炮台、坦克进行攻击，同时它还配备有展开的烟幕的能力，在战局或者战线僵持无法进行下去的时候，榴弹这种专门撕裂对方防御线的武器就变得非常厉害。 |                                                              |
| 巡航坦克           | 小队支援巡航坦克                                             | 具有运载整个小队的驾驶位，不仅是最基础的进攻用巡航坦克同时还可以投掷医疗弹药包支援友军。 |
| 迫击炮型巡航坦克   | 后置的炮手能使用迫击炮对远距离或者定点的敌人进行打击，同时后置炮手还可以发射出烟幕跟毒气炮弹，你可以认为这是一种用于反步兵的巡航坦克。 |                                                              |
| 坦克杀手型巡航坦克 | 由于设计是用来进行中距离反载具的，不但具有穿透力更强、伤害更高的大口径炮台（2、3号炮手位），还可以部署毒气烟幕来保护自己免受敌人干扰，同时还可以快速维修（1号驾驶专用），实为专业级反坦克载具。 |                                                              |
| 重型坦克           | 重型突击坦克                                                 | 除了1号位以外其他位置全是机枪的它适合用于中距离与步兵战斗，同时配有防御性烟幕等装备，维修起来也比其他两种重型坦克要快，很适合新人或者对某些地点进行压制性攻击。 |
| 重型突破坦克       | 这种重型坦克与其他不一样的是炮手的位置减少到了4位，但它的穿甲炮台（2、3号改为炮手座，驾驶主炮改为机枪）增加威力专门用于远距离攻击其他载具跟野战炮，除此以外还可以给友军提供医疗包跟弹药包补给（3、4号位专用），值得一提的是驾驶还可以使用紧急维修。 |                                                              |
| 烈焰重型战车       | 与前一种类型不一样的是，它同时配有了两座火焰喷射器，杀伤力跟精英兵种的喷火兵一样，在近距离巷战中显得非常具有杀伤力，而且还可以遮蔽敌人的视线，同时还能投射防御性的毒气烟幕，非常适合对步兵作战。 |                                                              |
| 火炮装甲车         | 火炮装甲车                                                   | 这是装有13磅炮弹以及反坦克地雷的装甲车，虽然开炮的时候无法移动并且需要切换武器，但也无法掩盖其他装甲单位对它的恐惧，毕竟它就是一台车载野战炮，不论对人还是装甲单位都可以造成可观的伤害。 |
| AA防空小车         | 从大炮改变为AA对空防御炮，这并不影响它给予人们的感观，换成了AA装备之后，它只不过把打击目标从地面改为空中，同时它还可以配置防御性烟幕来掩护自己，基本在空中飞的都是它的“猎物”。 |                                                              |
| 迫击炮装甲车       | 装载迫击炮的装甲车，但是装甲车的迫击炮可谓别具一格，因为它不止可以发射我们最常见的炮弹，还可以射击各种花样的炮弹，很合适用于清理远距离的敌兵。 |                                                              |

**飞机**

与以往不一样，飞机的操作发生了变化，以往玩家控制固定翼这类战斗机的时候需要油门、刹车、前翼转向、机身转向一起操作才可以勉强的飞得起来，而控制武装直升机、小鸟这类升降机的时候也需要爬升、降低、转向一起操作。

| 名称             | 扩展                                                         | 介绍                                                         |
| ---------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 攻击机           | 地面支援攻击机                                               | 这种配置的战斗机除了1、2号位的机枪以外1号位还可以投掷手雷炸弹来轰炸地面的步兵，在游戏中经常看见突然满屏的手雷图标就是来自手雷炸弹的。 |
| 坦克猎手攻击机   | 这种配置的攻击机直接把1号位的马克沁机枪换成了一门炮，对坦克跟飞机都可以造成很客观的伤害，一发可以打掉载具40血，但是由于炮弹下坠很严重，很考验飞行员的狙击水平。 |                                                              |
| 飞船毁灭者攻击机 | 这种飞机的特点可以发射《战地》系列曾经出现过的火蛇导弹，打起来跟烟花一样但是威力确实很惊人，对空对地都是一个很不错的选择。 |                                                              |
| 轰炸机           | 弹幕式轰炸机                                                 | 轰炸机的2、3号位都是配置有马克沁机枪，每种配置不一样的只有轰炸的炸弹，第一种炸弹就是万金油炸弹，什么都可以炸。 |
| 火焰式轰炸机     | 这种轰炸机的炸弹与前面的弹幕式轰炸机不一样，炸弹爆炸后会产生大量火焰对地面单位造成二次伤害，对付步兵是一种不错的选择。 |                                                              |
| 鱼雷式轰炸机     | 这种轰炸机的炸弹比较特殊，只能丢海上的小船，落入水面的鱼类会以直线向前飞驰，所以请在有海的地方再使用这种轰炸机。 |                                                              |
| 战斗机           | 空战战斗机                                                   | 最为纯粹的拦截与空战使用的战斗机，不但灵巧还具有快速修复功能，很适合用于空战狗斗跟猎杀其他飞机，当然你也可以用来猎杀地面的步兵。 |
| 轰炸机杀手战斗机 | 顾名思义它是一种专门为了对付轰炸机而诞生的战斗机，装备有射速比较慢的重型机炮，能对各种载具造成很客观的伤害，当然这需要看驾驶的技术，一般是用于猎杀轰炸机。 |                                                              |
| 战壕战斗机       | 其实这种战斗机与狗斗用的战斗机没什么区别，只是它比空中专用的多了一颗反步兵炸弹的选项，还可以发射信号弹用于标记敌人方向，反步兵的炸弹伤害非常可观，在一望无际的平地或者是战壕里面投掷一次两次总会炸死人。 |                                                              |

| M.A.S鱼雷快艇 | 作为《战地1》中唯一能随时随地使用的海上载具，它不仅配备有鱼雷还配备了几门机炮，其中2号位机炮是50mm的机炮，可以用于攻击岸上的坦克单位。 |
| L30空艇       | 在游戏中它一共可以乘坐7人，除了1号驾驶位以外2、3、5位使用的是50mm机炮，对地对空都是十分厉害的，但是炮手因为暴露在外，很容易因为炮火轰炸而阵亡；4、6位使用的马克沁机枪，是专门用来对付前来攻击飞艇的空中单位；1号位除了驾驶以外还允许玩家使用空投炸弹，与轰炸机不同的是，空艇的轰炸范围更广，伤害更高，甚至一颗下去还会把周边的房子炸毁。 |
| 装甲战列车    | 身为一战二战历史中的常客，这也是很多人所知的机械巨兽之一，但是由于其行驶需要根据铁路来行驶，DICE并没有把破坏系统加入到铁路上，毕竟你这样炸铁道那这个列车基本出来就废了，同时它也是能提供很强的火力来掩护步兵单位进行作战，车上搭载的炮台足以摧毁地面上的各种坦克。 |
| 无畏级战舰    | 它与其他巨兽不一样的是只可以乘坐四人，无畏级炮台是驾驶员兼职的，2、3号位使用的是比1号位稍微弱一点的炮台，同样对坦克还是步兵都可以造成很大的伤害，但是问题是它并不能射击船边的移动目标，比如鱼雷艇 |
